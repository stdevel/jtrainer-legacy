What is JTrainer?

JTrainer is a smart tool for training skills - it's an exam simulator. You can use it e.g. in preparation for an exam.

How to use catalogs

Catalogs have to be stored on the SD card in the directory data/jtrainer. To open a catalog, follow these steps:
  1. Run the JTrainer application.
  2. Tap on "Start".
  3. Tap on the input area (which contains "data/jtrainer/") and edit the path to your catalog file, e.g. "data/jtrainer/lpic101.txt".
  4. Tap on "Open file".
You might also add a catalog file to your favorites, so that you can open it quicker.

Favorites

To start your catalogs quicker, you can use the favorites list. You can choose the catalogs from a list and open them without entering the path and filename.

Add a catalog to favorites

To add a catalog to the list of favorites, follow these steps:
  1. Run the JTrainer application.
  2. Tap on "Favorites".
  3. Tap on the input area (which contains "data/jtrainer/") and edit the path to your catalog file, e.g "data/jtrainer/lpic101.txt".
  4. Click on the save icon to add the catalog to the favorites.

Remove a catalog from favorites

To remove a catalog to the list of favorites, do the following steps:
  1. Run the JTrainer application.
  2. Tap on "Favorites".
  3. Select the catalog from the list and click on the entry.
  4. Click on "Delete from favorites" to remove the catalog from the favorites list.

Open a catalog from favorites

To add a catalog to the list of favorites, do the following steps:
  1. Run the JTrainer application.
  2. Tap on "Favorites".
  3. Select the catalog from the list and click on the entry.
  4. Click on "Open" to open the catalog.

Delete a catalog from favorites

To completely delete a catalog, do the following steps:
  1. Run the JTrainer application.
  2. Tap on "Favorites".
  3. Select the catalog from the list and click on the entry.
  4. Click on "Delete file" to delete the catalog.

How to create your own catalogs

To create your own catalogs, you need a simple text edtior - you will have to write a file in a format as explained below. In future, there will be a catalog editor included in the software - so stay tuned for updates. ;-)
A catalog file has the following structure: 
		Number of questions
		Catalog name
		Catalog description
		Author 
		Date
		Time limit (if you don't want a time limit, enter 0 here)
		(Data records...)
A data record has the following structure: 
		Question
		Type
		Answer(s)
		Number of correct answer (if multiple, counting starts at 0!)
		Question hint (if you don't want a hint to be displayed, insert "NONE" here)
		Question explanation (if you don't want to display a explanation insert "NONE" here)
There are several question types:
  • 0 - only one answer, has to be typed using the keyboard
  • 1 - multiple answers, one is correct, displayed in a list
  • 2 - multiple answers, multiple are correct, displayed in a list

An example of a complete question catalog is: 
		1
		Test catalog
		This is a test catalog
		Christian Stankowic 
		23.01.2011
		1
		What's the product of 12*12?
		1
		153
		144
		192
		120
		112
		1
		The number is smaller than 180
		Everybody knows, that the product of 12*12 is 144!

Settings

There are two options, which can be configured in the settings dialog:
  • Random mode - Questions are asked randomly
  • Endless mode - The catalogs continues after finishing, questions are repeated until you stop the simulation
